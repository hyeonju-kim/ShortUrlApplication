# 미니프로젝트 ShortUrl Application 만들기

-   **사용 기술**

1.  spring web mvc
2.  spring data jpa
3.  h2 database
4.  thymeleaf

-   **화면 설명**

1.  원래의 url을 텍스트 창에 입력하고,  등록하기 버튼을 누르면 localhost:8080/srt1 과 같은 짧게 만들어진 url이 나타난다.
2.  짧은 url을 클릭해서 들어가면 원래의 url 로 redirect 되어 접속된다.
